# Project Suite Pro - Backend API

Complete backend API for Project Suite Pro construction management SaaS platform.

## Quick Start

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables:
```bash
cp .env.example .env
# Edit .env with your values
```

3. Set up database:
```bash
npx prisma generate
npx prisma migrate dev
```

4. Start server:
```bash
npm run dev
```

Server runs on http://localhost:5000
